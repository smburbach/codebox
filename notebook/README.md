# Lab Notebook

Structured record of project work, decisions, and results. Used by both humans and
coding agents to maintain continuity across sessions.

See `skills/lab-notebook/SKILL.md` in the parent [codebox](https://github.com/briney/codebox)
repo for full conventions and templates.

## Structure

```
notebook/
├── projects/        # Ongoing project directories
│   └── PROJECTS.md  # Index of active projects
└── experiments/     # One-off explorations and benchmarks
```

Each project directory contains:
- `STATUS.md` — current state, blockers, next steps
- `logs/` — timestamped session logs
